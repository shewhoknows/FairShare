# BillBandit Figma Atlas

Generated from the documented BillBandit screenshots in `docs/billbandit-workflow/screenshots`.

## Files

- `billbandit-screen-atlas.svg` - all screens in one Figma-importable board.
- `screens/*.svg` - one Figma-importable SVG per screen.
- `layer-map.json` - exact region names and coordinates for each overlay layer.

## Figma usage

1. Open the Figma file: https://www.figma.com/design/HouRHP9xkcfYEIxoLRwzMF
2. Drag `billbandit-screen-atlas.svg` into the canvas, or import individual files from `screens/`.
3. In Figma, each screen contains a screenshot backplate and named overlay regions. Use the region layers for comments or to mark redesign targets.

Note: the Figma MCP write API hit the Starter-plan tool-call limit after creating the file, so this atlas is prepared as a local import artifact.
